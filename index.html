Xali

Xali is a Web platform, a tool which allow to get together families. In countries where there are wars, lot of families are separated. If these people come in humanitarian organisation's camp, they'll can use Xali to add them in database, and the rest of family which is is an other country, will can find them with help of humanitarian organisation which will user Xali to find them.

What is Xali ?

In contries where there are wars, thousands of families are separated each years. Children lost their parents, and it's very difficult for organisations to get together them. Xali is a free and open source project which has for goal to get together lot of families who lives in these countries.

We stay in our country, but NGO, organisations, come into these countries in order to help population, we must help them in their task. This is why Xali is free and open source, every one can fork, use or participate to Xali, Xali has to help organisations.

Why create Xali ?

One day in a programming competition, a challenge was to imagine a Web platform which should help organisation to get together families. If this challenge was proposed, it's because there is a real need of this kind of platform. So, in order to help organisations, help families which are separated, we had to create Xali. It will take a long time until Xali be functional and useful, but with hard work and motivation Xali will help thousands of people.

How to contribute to Xali ?

There are several ways to contribute to Xali, no need to be a developper. If you're a Web designer, you can help to make the interface more intuitive, smarter or lighter. If you're a developper you can develop a functionnality or fix a bug. If you're a translator you can help to improve english strings, or help to translate Xali into your language. Else, if you want to contribute you can speak about Xali, on social networks for example.
